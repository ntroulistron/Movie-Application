About This Movie Application

Welcome to the Movie Application! This web app allows you to explore a wide range of movies, including the latest releases, top-rated films, and much more. By integrating with The Movie Database (TMDb) API, we provide up-to-date information on movies, including details such as genres, ratings, release dates, and movie posters.

Whether you're a casual movie watcher or a dedicated cinephile, this application makes it easy to discover new movies, search for specific titles, and even browse by genres. With a user-friendly design and responsive features, this website ensures that users can seamlessly explore movies across various devices.

Key Features:

  • Search movies by title
  • Browse by genre using TMDb data
  • View movie details, including posters, release dates, and ratings
  • Dark and Light theme modes for user preference
  • Mobile-responsive design
